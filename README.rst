*********
Formattie
*********

A simple tool to format and syntax highlight ugly XML code.

Useful for pasting XML-without-linebreaks from Firebug.


=================
Supported formats
=================

- XML
- Serialized PHP variable
- JSON
- URLs


=====
Setup
=====
Install geshi::

  $ composer install
